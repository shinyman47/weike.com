# weike.com
在线网课学习网站/学生毕业设计作品

# 微课网站0.8测试版本

1.用户免费注册账号，在线搜索学习视频网课

2.注册用户对网课进行评价，打分，并审核后展示在课程主页面

3.课程暂由管理员上传，用户不得自行上传私人课程

4.学生学习每节课程均有五道课后测试供温习课程
